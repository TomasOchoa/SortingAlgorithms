## Bubble Sort

### Performance

- Worst-case: **`O(n²)`**
- Best-case: **`O(n)`**
- Average-case: **`O(n²)`**

### Space Complexity

- Worst-case: **`O(1)`** auxillary

### Abstract

- Compare each item to the next item.
- Swap if required.
- Repeat until a pass without any swaps occurred
    - No swaps `==` sorted

### Example

> **7**,**5**,2,4,3,9

> 5,**7**,**2**,4,3,9

> 5,2,**7**,**4**,3,9

> 5,2,7,**4**,**3**,9

> 5,2,7,4,**3**,**9**

> 5,2,7,4,3,9
